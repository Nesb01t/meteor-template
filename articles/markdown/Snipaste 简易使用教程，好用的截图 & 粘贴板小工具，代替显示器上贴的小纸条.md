# 常用功能

被安利是因为 Snipaste 有一个功能，可以把截到的图置顶，粘在你的显示器上，等于在显示器上粘了一个小纸条，在日常工作 / 游戏中用途广泛。

我用的是免费版，同样是几乎所有功能都有了。

当你按下 Ctrl + Alt + A (由于常用QQ的习惯，我自己改的按键)，框选需要的区域以后，你就可以选择：

*   钉在显示器上
*   复制 (也可以通过快捷键Ctrl+C)
*   保存到指定位置
